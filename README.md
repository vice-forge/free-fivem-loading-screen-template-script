## 🛠️ Manual Installation (For Developers)
If you prefer not to use the visual builder at Vice Forge and want to code it yourself:
1. Download the code from this repository.
2. Extract the folder into your FiveM `resources` directory.
3. Open your `server.cfg` and add `ensure free-fivem-loading-screen-script`.
4. Edit the `index.html` and `style.css` files manually to change text and images.
